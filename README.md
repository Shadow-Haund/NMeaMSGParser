# NMeaMSGParser
Parser for GGA, GLL, GSA, GSV, RMC, VTG, GST, ZDA, DTM and GNS messages

# To run this code
Code was writen under Ubuntu. To run this code copy it on your computer and in terminal go to src directory, then execute following command

    g++ main.cpp gnss_parser.cpp -o a.out
    
and then 

    ./a.out

Execution result will be saved inside **data/output.txt** file
